User Management

Admins can manage system users, assign roles, and control access permissions.

User Roles

Role Access Level
Admin Full system access. Can manage users, settings, and all modules.
Coordinator Can manage bookings, guides, and view reports. No access to system settings or user management.
Guide Can view assigned tours and update their own profile/availability.
Security Access to Check-in/out and Incident reporting only.
Visitor Can only view and manage their own bookings.

Managing Users

Creating a User

  1. Navigate to Users.
  2. Click Add User.
  3. Enter email, password, and personal details.
  4. Select the appropriate Role.
  5. Click Create User.

Editing a User

  • Change Role: Promote or demote a user's access level.
  • Deactivate: Disable a user account without deleting data.
  • Reset Password: Trigger a password reset email.

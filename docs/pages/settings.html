Settings

Configure global system behavior, pricing, and preferences.

Pricing Configuration

Set the base prices for different tour types and group sizes.

Group Size Base Price (MWK) Description
Individual 15,000 1 person
Small Group 50,000 2-5 people
Large Group 80,000 6-10 people
Custom 100,000 10+ people or special requests

Additional Costs

  • Extra Hour: 10,000 MWK per hour beyond standard duration.

System Preferences

  • Currency: Default currency for display (MWK).
  • Timezone: System timezone setting.

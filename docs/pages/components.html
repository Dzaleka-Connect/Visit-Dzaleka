UI Components

Visit Dzaleka uses shadcn/ui, a collection of re-usable components built with Radix UI and Tailwind CSS.

Core Components

Components are located in client/src/components/ui.

Button

import { Button } from "@/components/ui/button"

Card

import { Card, CardContent, CardHeader, CardTitle } from "@/components/ui/card"


  
    Card Title
  
  
    
Card content goes here.

Dialog (Modal)

import { Dialog, DialogContent, DialogTrigger } from "@/components/ui/dialog"


  Open
  
    
Modal Content

Custom Components

Application-specific components located in client/src/components.

StatCard

Displays a dashboard statistic with an icon and trend indicator.

StatusBadge

Displays a colored badge for booking statuses.

CLI Commands

Available npm scripts for development and maintenance.

Development

npm run dev

Starts the development server with hot reloading.

  • Frontend: Vite dev server
  • Backend: tsx watch mode
  • URL: http://localhost:5000

npm run build

Builds the application for production.

  • Compiles TypeScript
  • Bundles frontend assets with Vite
  • Bundles backend with esbuild
  • Output: dist/ directory

npm start

Starts the production server from the dist/ directory.

Database

npm run db:push

Pushes schema changes to the database using Drizzle Kit.

Warning: This may modify your database schema. Always backup production data first.

npm run check

Runs TypeScript type checking across the entire project.

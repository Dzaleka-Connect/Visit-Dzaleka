Environment Variables

Configuration variables for the .env file.

Variable Required Description
DATABASE_URL Yes PostgreSQL connection string (e.g., postgresql://user:pass@host/db)
SESSION_SECRET Yes Secret key for signing session cookies. Use a long random string.
RESEND_API_KEY No API key for Resend email service. Required for sending emails.
NODE_ENV No development or production. Defaults to development.
PORT No Server port. Defaults to 5000.

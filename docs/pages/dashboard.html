Dashboard Overview

The Dashboard is the central hub of Visit Dzaleka, providing a real-time overview of camp visit activities, revenue, and key performance metrics. The view is customized based on your user role.

Charts & Analytics

Weekly Booking Trends

A bar chart showing the number of bookings for each day of the current week. This helps in identifying peak visit days and planning resource allocation.

Popular Zones

A pie chart displaying the distribution of visits across different camp zones (e.g., Market, Community Center). Use this to understand which areas are most attractive to visitors.

Guide Performance

A comparative bar chart showing the number of tours completed by each guide. This is useful for balancing workloads and recognizing top performers.

Quick Actions

The dashboard provides one-click access to common tasks:

  • New Booking: Create a manual booking request.
  • Add Guide: Onboard a new tour guide.
  • Check In: Quickly check in a visitor (Security role).

Recent Activity

A feed of the latest system activities, including:

  • New booking requests
  • Booking confirmations
  • Visitor check-ins/check-outs
  • New user registrations

Security Module

The Security module is designed for security personnel to manage visitor access, verify bookings, and handle incidents.

Visitor Check-In/Out

Check-In Process

  1. Visitor arrives at the gate.
  2. Security searches for the booking by name or reference ID.
  3. Verify visitor identity.
  4. Click Check In.
  5. The booking status updates to "In Progress".

Check-Out Process

  1. Visitor returns to the gate after the tour.
  2. Locate the active booking.
  3. Click Check Out.
  4. The booking status updates to "Completed".

Incident Reporting

Log and track security incidents that occur within the camp.

Reporting an Incident

  1. Go to the Incidents tab.
  2. Click Report Incident.
  3. Select Severity (Low, Medium, High, Critical).
  4. Describe the incident details.
  5. Submit the report.

Managing Incidents

  • Update Status: Change from Reported → Investigating → Resolved.
  • Add Notes: Append additional information to an open case.

Active Visitors

The Active Visitors tab shows a real-time list of all visitors currently checked into the camp, along with their assigned guides and expected duration.

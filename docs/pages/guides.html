Guide Management

The Guides module allows you to manage your team of tour guides, track their performance, and handle tour assignments.

Managing Guide Profiles

Adding a New Guide

  1. Navigate to the Guides page.
  2. Click Add Guide.
  3. Enter personal details (Name, Phone, Email).
  4. Availability: Select days of the week and preferred times.
  5. Payment: Choose preferred payment methods (Cash, Mobile Money).
  6. Emergency Contact: Add contact details for emergencies.
  7. Click Save Guide.

Guide Status

Guides can be toggled as Active or Inactive. Inactive guides will not appear in the assignment list for new bookings but their history is preserved.

Guide Performance

Navigate to Guide Performance to view analytics:

Earnings Chart

View monthly earnings for each guide to track payouts and revenue generation.

Leaderboard

See top-performing guides based on tour count, completion rate, and visitor ratings.

Availability Calendar

Visual calendar showing scheduled tours for all guides to prevent double-booking.

Assigning Guides

To assign a guide to a booking:

  1. Go to the Bookings page.
  2. Find a confirmed booking.
  3. Click Assign Guide.
  4. Select an available guide from the list.
  5. The guide will receive a notification (if email is configured).

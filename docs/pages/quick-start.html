Quick Start Guide

This guide will help you get up and running with Visit Dzaleka in minutes.

1. Create an Admin Account

After installation, the first step is to create an administrator account. Since there are no users initially, you'll need to create the first user via the API or database.

Note: In a production environment, you would typically seed the database with an initial admin user.

Using the API (Development Mode)

You can use the registration endpoint to create your first user:

curl -X POST http://localhost:5000/api/auth/register \
  -H "Content-Type: application/json" \
  -d '{
    "email": "admin@dzaleka.com",
    "password": "securePassword123",
    "firstName": "Admin",
    "lastName": "User"
  }'

By default, new registrations are assigned the visitor role. You'll need to upgrade this user to admin in the database.

Upgrading to Admin Role

Run the following SQL command in your database:

UPDATE users SET role = 'admin' WHERE email = 'admin@dzaleka.com';

2. Log In to the Dashboard

  1. Navigate to http://localhost:5000
  2. Enter your email and password
  3. Click Login

Upon successful login, you will be redirected to the Admin Dashboard.

3. Configure System Settings

Before managing bookings, configure the basic system settings:

  1. Go to Settings in the sidebar
  2. Review the Pricing Configuration
  3. Set up Email Templates under Operations

4. Set Up Camp Zones

Define the areas within the camp that visitors can tour:

  1. Navigate to Zones & POI
  2. Click Add Zone
  3. Create zones like "Market Area", "Community Center", "Education Hub"
  4. Add Points of Interest to each zone

5. Onboard Guides

Add tour guides to the system:

  1. Navigate to Guides
  2. Click Add Guide
  3. Enter guide details (Name, Phone, Languages)
  4. Set their availability and payment preferences

6. Create a Test Booking

Verify the system is working by creating a manual booking:

  1. Go to Bookings
  2. Click New Booking
  3. Fill in visitor details and tour preferences
  4. Submit the booking

You should see the new booking in the "Pending" list.

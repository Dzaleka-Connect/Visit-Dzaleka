Email System

Visit Dzaleka includes a robust email notification system powered by Resend to keep visitors and staff informed.

Email Templates

Admins can manage email templates via the Email Templates page.

Available Templates

The system sends the following automated emails:

  • Booking Confirmation: Sent to visitor when a booking request is received (Pending).
  • Status Update: Sent when booking status changes (Confirmed, Cancelled, Completed).
  • Guide Assignment: Sent to visitor when a guide is assigned to their tour.
  • Check-In Notification: Sent to visitor upon successful security check-in.
  • Password Reset: Sent to users requesting a password reset.

Template Variables

Each template uses specific variables to personalize the content:

Booking Confirmation

  • visitorName - Name of the visitor
  • bookingReference - Unique booking ID
  • visitDate & visitTime - Scheduled date and time
  • tourType - Type of tour selected
  • numberOfPeople - Group size
  • totalAmount - Calculated cost
  • meetingPoint - (Optional) Selected meeting location

Status Update

  • newStatus - The updated status (Confirmed, Completed, Cancelled)
  • adminNotes - (Optional) Message from the administrator
  • Includes all Booking Confirmation variables

Guide Assignment

  • guideName - Name of the assigned guide
  • guidePhone - Contact number for the guide
  • Includes all Booking Confirmation variables

Check-In Notification

  • checkInTime - Timestamp of check-in
  • guideName - (Optional) Name of the guide meeting the visitor

Notifications

The system automatically sends emails for:

  • New booking requests (to Coordinators)
  • Booking status updates (to Visitor)
  • Guide assignments (to Guide)
  • Password resets (to User)

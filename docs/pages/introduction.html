Introduction to Visit Dzaleka

Visit Dzaleka is an enterprise visitor management system designed specifically for Dzaleka Refugee Camp in Malawi, Africa's fourth-largest refugee settlement.

What is Visit Dzaleka?

Visit Dzaleka is a comprehensive web-based platform that enables administrators to manage tour bookings, coordinate guides, oversee camp zones and points of interest, handle visitor scheduling, manage security check-in/out, report incidents, and control multi-role access through an intuitive dashboard interface.

The application serves as a central booking and coordination platform for organizing guided tours of the refugee camp, supporting:

  • Multiple tour types - Standard, Extended, and Custom tours
  • Flexible group sizes - Individual, Small Group, Large Group
  • Local payment methods - Cash, Airtel Money, TNM Mpamba
  • Email notifications - Automated confirmations and reminders
  • Security protocols - Check-in/out tracking and incident reporting

User Roles

Visit Dzaleka supports five distinct user roles, each with tailored permissions and dashboard views:

Role Description Key Permissions
Admin Full system administrator All features, user management, system settings
Coordinator Booking and tour coordinator Manage bookings, assign guides, view reports
Guide Tour guide View assigned tours, update tour status
Security Security personnel Check-in/out, incident reporting, verification
Visitor Tour visitor View own bookings, submit booking requests

Technology Stack

Visit Dzaleka is built with modern web technologies:

  • Frontend: React 18, TypeScript, Tailwind CSS, Radix UI
  • Backend: Node.js, Express.js, TypeScript
  • Database: PostgreSQL with Drizzle ORM
  • Authentication: Session-based with bcrypt password hashing
  • Email: Resend API for transactional emails

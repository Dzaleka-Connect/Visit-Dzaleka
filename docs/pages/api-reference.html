API Reference

The Visit Dzaleka API is a RESTful interface built with Express.js. All endpoints are prefixed with /api.

Authentication

POST /api/auth/login

Authenticate a user and create a session.

POST /api/auth/logout

Destroy the current session.

GET /api/user

Get the currently authenticated user.

Bookings

GET /api/bookings

List all bookings. Supports filtering by status and date.

POST /api/bookings

Create a new booking request.

PATCH /api/bookings/:id/status

Update booking status (e.g., confirm, cancel).

Guides

GET /api/guides

List all guides.

POST /api/guides

Create a new guide profile.

GET /api/stats/guide-performance

Get performance metrics for guides.

Zones

GET /api/zones

List all camp zones.

POST /api/zones

Create a new zone.

Troubleshooting

Common issues and their solutions.

Database Connection Issues

Error: Connection refused or terminating connection due to administrator command

Solution:

  • Check if your IP is allowed in the database provider's settings (e.g., Supabase/Neon).
  • Verify the DATABASE_URL in your .env file.
  • If using a local DB, ensure the PostgreSQL service is running.

Session Errors

Error: Error: secret option required for sessions

Solution:

  • Ensure SESSION_SECRET is defined in your .env file.

Build Failures

Error: TypeScript error: Property 'x' does not exist on type 'y'

Solution:

  • Run npm run check to identify all type errors.
  • Ensure your local schema matches the database schema.

Managing Bookings

The Bookings module is the core of the Visit Dzaleka system, allowing you to manage the entire lifecycle of a tour reservation.

Booking Lifecycle

1

Pending

New booking request received. Requires review and confirmation.

2

Confirmed

Booking approved. Guide assigned (optional but recommended).

3

In Progress

Visitor has checked in and the tour is currently happening.

4

Completed

Tour finished successfully. Visitor has checked out.

Creating a Booking

Admins and Coordinators can create bookings manually:

  1. Navigate to the Bookings page.
  2. Click the New Booking button.
  3. Fill in the visitor details (Name, Email, Phone).
  4. Select tour preferences (Date, Time, Group Size, Tour Type).
  5. Click Create Booking.

Managing Existing Bookings

Filtering and Search

Use the toolbar to find specific bookings:

  • Search: Find by visitor name or email.
  • Status Filter: View only Pending, Confirmed, etc.
  • Date Range: View bookings for a specific period.

Booking Actions

Click the actions menu (three dots) on any booking row to:

  • View Details: See full booking information.
  • Edit: Modify booking details.
  • Confirm/Cancel: Change booking status.
  • Assign Guide: Select a guide for the tour.

Record Past Visit

To log a visit that has already occurred (for record-keeping):

  1. Click Record Past Visit on the Bookings page.
  2. Enter all visit details including the completed date.
  3. The booking will be created with "Completed" status automatically.

Exporting Data

You can export booking data for external analysis:

  1. Filter the list to show the data you want to export.
  2. Click the Export CSV button.
  3. The file will download automatically.

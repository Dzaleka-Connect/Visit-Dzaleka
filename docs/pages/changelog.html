Changelog

History of changes and updates to the Visit Dzaleka application.

v1.0.0 (December 2024)

Initial release of the comprehensive visitor management system.

Features

  • Authentication: Secure email/password login with role-based access control.
  • Dashboard: Role-specific dashboards with real-time analytics.
  • Bookings: Full booking lifecycle management (Pending → Completed).
  • Guides: Guide profiles, availability tracking, and performance metrics.
  • Security: Visitor check-in/out and incident reporting.
  • Email: Automated notifications via Resend.
  • Zones: Management of camp zones and points of interest.

Technical

  • Migrated to React 18 + Vite.
  • Implemented Drizzle ORM with PostgreSQL.
  • Added comprehensive audit logging.
